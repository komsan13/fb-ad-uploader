# ส่วนขยาย "สลับบัญชี FB" — คู่มือติดตั้ง

ส่วนขยายนี้ทำให้ปุ่ม **เปิดบัตร / เช็คยอด** ในเครื่องมือ เด้งเข้า **บัญชี Facebook ถูกตัว** ให้อัตโนมัติ
ไม่ต้องมานั่งสลับล็อกอินเองทีละบัญชี

## มันทำงานยังไง (สั้นๆ)

- **เชื่อมบัญชี (ทำครั้งเดียวต่อบัญชี):** คุณล็อกอิน facebook.com ด้วยตัวเอง (ผ่าน 2FA แบบปกติ)
  แล้วกด "เชื่อมบัญชีนี้" ในการ์ดบัญชี → ส่วนขยายจะจำ session ของบัญชีนั้นไว้
- **เปิดบัตร/เช็คยอด (กดกี่ครั้งก็ได้):** ส่วนขยายสลับ cookie ของ facebook.com เป็นบัญชีนั้นให้ แล้วเปิดหน้าบิล

**ปลอดภัย:** ไม่เก็บรหัสผ่าน ไม่ล็อกอินแทนคุณ ไม่ยิงจากเซิร์ฟเวอร์ —
session ได้จากการที่คุณล็อกอินเองในเบราว์เซอร์นี้ และเก็บไว้ในเบราว์เซอร์นี้เท่านั้น (ไม่ส่งขึ้นเซิร์ฟเวอร์)

## ติดตั้ง (Chrome / Edge)

1. เปิด `chrome://extensions` (Edge: `edge://extensions`)
2. เปิดสวิตช์ **โหมดนักพัฒนา (Developer mode)** มุมขวาบน
3. กด **โหลดส่วนขยายที่แตกไฟล์แล้ว (Load unpacked)** → เลือกโฟลเดอร์ `extension/` นี้
4. เสร็จ — กลับไปที่หน้าเครื่องมือแล้วรีเฟรช จะเห็นปุ่ม "เชื่อมบัญชีนี้" ในแต่ละการ์ดบัญชี FB

## วิธีใช้

1. หน้าบัญชี FB → เลือกการ์ดบัญชีที่ต้องการ
2. เปิดแท็บใหม่ไป facebook.com แล้วล็อกอิน **บัญชีนั้น** (ถ้ายังไม่ได้ล็อกอิน)
3. กลับมาที่การ์ด กด **"เชื่อมบัญชีนี้ (สลับอัตโนมัติ)"** → ขึ้น "เชื่อม session แล้ว (id …)"
4. จากนี้ปุ่ม **เปิดบัตร** ในหน้า "สุขภาพบัญชี" จะสลับเข้าบัญชีนั้นให้เอง

> ถ้าเปลี่ยนรหัส/ออกจากระบบบัญชีนั้น session จะหมดอายุ — แค่ล็อกอินใหม่แล้วกด "เชื่อมบัญชีนี้" อีกครั้ง

## หมายเหตุ

- ใช้ได้เฉพาะเบราว์เซอร์ตระกูล Chromium (Chrome, Edge, Brave)
- รองรับเฉพาะโดเมน `https://ad.senball.com` (ตั้งไว้ใน `manifest.json` ที่ `content_scripts.matches` + `host_permissions`)
  **จงใจไม่ใส่ `localhost`** เพื่อกันหน้าเว็บ localhost อื่นในเบราว์เซอร์เดียวกันสั่งส่วนขยายให้สลับ/ลบ session ได้
  ถ้าต้องเทสบนเครื่อง ให้เพิ่ม origin ของ dev (เช่น `http://localhost/*`) ทั้งใน `content_scripts.matches` และ `host_permissions` ชั่วคราว แล้วโหลดส่วนขยายใหม่ — เอาออกก่อนใช้งานจริง
