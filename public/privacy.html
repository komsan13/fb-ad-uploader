1. ภาพรวม

แอปพลิเคชันนี้เป็นเครื่องมือภายใน (internal tool) สำหรับสร้างและจัดการแคมเปญโฆษณาบน Facebook ผ่าน Meta Marketing API ใช้งานโดยทีมงานที่ได้รับอนุญาตเท่านั้น ไม่เปิดให้บุคคลทั่วไปใช้งาน

2. ข้อมูลที่เก็บ

  • Access token ของบัญชี Facebook ที่ผู้ใช้เชื่อมต่อเอง เพื่อเรียกใช้ Marketing API ในนามผู้ใช้
  • รายชื่อบัญชีโฆษณา เพจ Pixel และสถิติแคมเปญ ที่ดึงจาก API เพื่อแสดงผลเท่านั้น
  • ไฟล์ภาพ/วิดีโอที่ผู้ใช้อัปโหลดเพื่อสร้างโฆษณา (ส่งต่อไปยัง Facebook และไม่เก็บถาวรบนเซิร์ฟเวอร์)

3. การใช้ข้อมูล

ข้อมูลทั้งหมดถูกใช้เพื่อวัตถุประสงค์เดียว: สร้างและจัดการโฆษณาตามคำสั่งของผู้ใช้ ไม่มีการขาย แบ่งปัน หรือส่งต่อข้อมูลให้บุคคลที่สาม

4. การเก็บรักษาและความปลอดภัย

ข้อมูลการเชื่อมต่อเก็บอยู่บนเซิร์ฟเวอร์ส่วนตัวที่จำกัดการเข้าถึงด้วยรหัสผ่าน ผู้ใช้สามารถลบบัญชีที่เชื่อมต่อออกจากระบบได้ตลอดเวลา ซึ่งจะลบ token ออกจากเซิร์ฟเวอร์ทันที

5. การลบข้อมูล

หากต้องการให้ลบข้อมูลที่เกี่ยวข้องกับบัญชีของคุณ ติดต่อผู้ดูแลระบบ หรือถอนสิทธิ์แอปได้เองที่ การตั้งค่า Facebook → แอปและเว็บไซต์
