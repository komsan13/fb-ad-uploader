ขยายงบ: ใส่ 0 = ไม่ขยาย • ถ้าใส่เลข แคมเปญที่ต้นทุน/ผลลัพธ์ถูกกว่าเพดานที่ตั้งไว้ 30%+ (และมีผลลัพธ์อย่างน้อย 3 ครั้ง) จะถูกเพิ่มงบ +20% วันละครั้ง จนถึงเลขนี้แล้วหยุด — เพิ่มทีละน้อยเพราะขยับแรงกว่านี้ FB รีเซ็ต learning phase แล้วผลตก

สิ่งที่มันทำ: อ่านเหตุผลที่ FB ปฏิเสธ → ถ้าปัญหาอยู่ที่ข้อความและระบุได้ชัด จะแก้แล้วขึ้นใหม่ให้ ครั้งเดียวต่อครีเอทีฟ • ถ้าปัญหาอยู่ที่วิดีโอ/หน้าขาย/ระดับบัญชี จะไม่แตะ แค่เตือนคุณ
ปิดตัวขาดทุนเอง: แอดที่ใช้เงินเกิน 2 เท่าของเพดาน CPA แล้วยังไม่มีผลลัพธ์ หรือต้นทุนแพงกว่าเพดาน 1.5 เท่า จะถูกปิด (ไม่เกิน 10 ตัว/บัญชี/วัน) แล้วเติมครีเอทีฟใหม่เข้าไปแทน — ใช้เลขจากช่อง "หยุดเมื่อต้นทุน/ผลลัพธ์เกิน" ในหน้าขึ้นแอด ถ้าไม่ได้ตั้งไว้จะไม่ปิดอะไรเลย
เพดานกันพัง: แก้ได้ไม่เกิน 10 ครั้ง/วันทั้งระบบ • บัญชีไหนโดนปฏิเสธ 3 ตัวใน 24 ชม. จะหยุดบัญชีนั้นทันทีเพื่อกันโดนแบน • แจ้งทุกอย่างเข้า Telegram
ปรับเพดานเอง
ทุกช่องมีกรอบต่ำสุด–สูงสุดบังคับไว้ ใส่นอกกรอบระบบจะบีบกลับให้เอง เพราะค่าพวกนี้คือเกราะกันบัญชีโดนแบนและกันเงินไหล
เว้นว่างหรือกดปุ่มคืนค่าตั้งต้น = กลับไปใช้ค่าเดิมของระบบ • ทุกการแก้ถูกจดไว้ในบันทึกด้านล่าง
วิธีหยุดระบบคือปุ่มในหน้านี้เท่านั้น — ระบบถูกตั้งให้ดูแลตัวเองเต็มที่ ถ้าไม่มีแคมเปญที่เปิดอยู่ มันจะสร้างใหม่ให้เสมอ ดังนั้นการเข้าไป pause แคมเปญใน Ads Manager จะไม่ได้ผล เพราะมันจะสร้างตัวใหม่มาแทนภายในไม่กี่นาที
จะหยุดจริงต้องปิดสวิตช์ "เปิดระบบอัตโนมัติ" หรือกด "หยุดฉุกเฉิน" ที่นี่

ตั้งค่า "เข้าสู่ระบบด้วย Facebook" — ทำครั้งเดียวต่อแอป หลังจากนั้นกดปุ่มเดียวจบ

  1. เปิดแอปที่ developers.facebook.com/apps → "เพิ่มผลิตภัณฑ์" → เพิ่ม Facebook Login (Login ด้วย Facebook)
  2. ไปที่ Facebook Login → การตั้งค่า (Settings) → ช่อง Valid OAuth Redirect URIs วาง: http://localhost:4000/auth/callback → บันทึก
  3. เมนู การตั้งค่าแอป → พื้นฐาน (Basic) → คัดลอก App ID และ App Secret (กด "แสดง") มาวางในการ์ดบัญชีด้านบน
  4. กดปุ่ม เข้าสู่ระบบด้วย Facebook → login → กดยอมรับ → เสร็จ! token มาเองและต่ออายุ 60 วันอัตโนมัติ
บัญชี FB ที่ login ต้องเป็นแอดมิน/ผู้ทดสอบของแอปนั้น (เฟสที่สร้างแอปเองเป็นแอดมินอยู่แล้ว ไม่ต้องส่ง App Review) • ถ้าหลาย ad account อยู่ใต้ FB เดียวกัน login ครั้งเดียวเห็นครบทุกบัญชี
