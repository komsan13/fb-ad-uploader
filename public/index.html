บัญชี FB ทั้งหมด — เพิ่มได้หลายบัญชี แต่ละบัญชีมี token + บัญชีโฆษณา + เพจของตัวเอง

แจ้งเตือน Telegram — เตือนทันทีเมื่อแอดโดนปฏิเสธ / บัญชีถูกปิด-ค้างชำระ / token มีปัญหา (ต่ออายุให้อัตโนมัติ) + สรุปยอดทุกเช้า 8 โมง

วิธีตั้ง (ครั้งเดียว): ① ใน Telegram แชทกับ @BotFather → พิมพ์ /newbot → ได้ Bot Token มาวางช่องแรก ② แชทกับ @userinfobot → ได้เลข Id มาวางช่องสอง ③ เข้าไปกด Start ในแชทบอทที่เพิ่งสร้าง ④ กดบันทึก → ส่งทดสอบ

ตั้งค่า "เข้าสู่ระบบด้วย Facebook" — ทำครั้งเดียวต่อแอป หลังจากนั้นกดปุ่มเดียวจบ

  1. เปิดแอปที่ developers.facebook.com/apps → "เพิ่มผลิตภัณฑ์" → เพิ่ม Facebook Login (Login ด้วย Facebook)
  2. ไปที่ Facebook Login → การตั้งค่า (Settings) → ช่อง Valid OAuth Redirect URIs วาง: http://localhost:4000/auth/callback → บันทึก
  3. เมนู การตั้งค่าแอป → พื้นฐาน (Basic) → คัดลอก App ID และ App Secret (กด "แสดง") มาวางในการ์ดบัญชีด้านบน
  4. กดปุ่ม เข้าสู่ระบบด้วย Facebook → login → กดยอมรับ → เสร็จ! token มาเองและต่ออายุ 60 วันอัตโนมัติ
💡 บัญชี FB ที่ login ต้องเป็นแอดมิน/ผู้ทดสอบของแอปนั้น (เฟสที่สร้างแอปเองเป็นแอดมินอยู่แล้ว ไม่ต้องส่ง App Review) • ถ้าหลาย ad account อยู่ใต้ FB เดียวกัน login ครั้งเดียวเห็นครบทุกบัญชี
